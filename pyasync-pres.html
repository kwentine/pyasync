2 Le serveur Fibonacci et ses clients

2.1 Les nombres de Fibonacci

  • Algorithme récursif
def fib(n):
    """Compute n-th Fibonacci number"""
    if n <= 1: return n
    else: return fib(n-1) + fib(n-2)
